Dispositivos de Rede

Aqui estão os principais dispositivos de rede e suas funções.

1. Switch

O switch conecta dispositivos dentro de uma rede e encaminha os pacotes de dados de forma eficiente.

2. Hub

O hub transmite dados para todos os dispositivos conectados, sem distinguir os destinatários.

3. Roteador

O roteador conecta redes diferentes e distribui a internet para múltiplos dispositivos.

4. Modem

O modem converte sinais digitais e analógicos para permitir acesso à internet.

5. Repetidor

O repetidor amplia o sinal de rede para cobrir uma área maior.

6. Bridge (Ponte)

A bridge conecta duas redes separadas, permitindo a comunicação entre elas.
